Static: The element is positioned to the normal flow of the document


Relative: the element is positioned relative to its normal position. setting the top, right, bottom and left properties will cause it to be adjusted away from its normal position.


Absolute: the element is positioned relative to its closest positoined ancestor (not static). If there are no such ancestors, it uses the document body, and moves along with page scrolling


Fixed: the element is positioned relative to the viewpoint which means it always stays in the same place even if the page is scrolled.


Sticky: the element is treated as relative positioned until it crosses a specified threshold, at which point it is treated as fixed positioned.
